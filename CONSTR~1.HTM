Automation

Weekly Auto-Sync (Every Friday)

A web page like this Dashboard cannot reach into a folder on your PC (e.g. C:\Users\HPVICTUS\Desktop\CM) by itself — browsers are blocked from doing that for security reasons, and there's no way to run a hidden background job on your computer from a webpage. To get a real weekly auto-update, you need two small pieces working together:

  1. A tiny script on your PC (Windows Task Scheduler, runs every Friday) that takes the latest Excel file from your CM folder and uploads it to one fixed, public link.
  2. This Dashboard checks that same fixed link automatically every time it's opened, and loads whatever the latest file is — no manual upload needed.

Use the box below to test a hosted file link right now. To make the Dashboard automatically load that link every time anyone opens it (true "auto-sync"), the link needs to be saved permanently inside this Dashboard file — send me the working link and I'll bake it in and give you an updated file to redeploy. After that, no one ever needs to upload manually again; it'll always show whatever your Friday script last uploaded.

Petty Cash (THB) = the project's allocated petty cash float · Collected (THB) = payment already collected from the client/owner · Man Power (Hours) = cumulative site-inspection hours · Site Visit Days = number of days a site visit was conducted · Site Inspector = name of the person who carried out the site visit · Preliminary = preliminary/general site cost item · VO = Variation Order, value of additional work · VE = Variation/Omission, value of reduced or omitted work · Contractor Name = the main contractor engaged for the project (use "Not Yet Awarded" if not yet contracted) · Contract Value = the awarded contract sum for that contractor

The Status column accepts: Construction / Completed / Planning / On Hold — Project names in the Master Schedule must match exactly with names in the Summary Report so the system can correctly link the two sheets.

Sheet: Site Visit Log (optional — 1 row = 1 team member's visits in 1 month)

Powers the Monthly Site Visit Overview chart — a team-wide view of off-site work execution by month, in hours (not tied to a specific project). Add one row per team member per month. Month can be any date within that month (e.g. the 1st). If this sheet is missing, the chart will simply show no data.
